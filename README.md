forex-sample-dataset
====================

Forex sample dataset of time-series candles

Data files:

```
<data_type>.<interval>.<currency_pair>.csv
```

- `<data_type>`:
  - `candle`: historical prices
- `<interval>`:
  - `S`: second
  - `M`: minute
  - `H`: hour
  - `D`: day
- `<currency_pair>`:
  - `EUR_USD`: EUR/USD
